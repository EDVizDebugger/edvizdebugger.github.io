Examples

After installing EDViz, you can download the examples below to get started with the tool. All of these examples consist of an Eclipse project in a folder named project and a configuration file for EDViz at the same level. Just download any of them, import the project into Eclipse and load the configuration file within the example. Then set breakpoints where desired and debug the Main class.
