Program it. Debug it. Visualize it.

Download

EDViz bridges the gap between data structure teaching and data structure implementation. Visualize your data structures not only on the blackboard, but also your computer screen. EDViz is an Eclipse plug-in which visualizes your data structures when you debug them. Install the tool, open it and read the tutorial to get started. For any further questions, take a look at the manual. And most importantly, just use it!

Requires Eclipse Neon or Eclipse Mars on any operating system.
